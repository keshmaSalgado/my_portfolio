## 3D Portfolio Website

An interactive 3D portfolio built using Three.js and Blender to showcase my projects, skills, and creative work.

The website features real-time 3D graphics, animated models, and a modern responsive design.

### Features

- Interactive 3D scene
- Blender 3D models
- Smooth camera animations
- Responsive modern UI
- Project showcase section
- Real-time WebGL rendering

### Tech Stack

Frontend
- JavaScript / React
- Three.js
- React Three Fiber
- HTML / CSS

3D Tools
- Blender
